LINKTYPE_ZWAVE_R3

Packet structure

+---------------------+
|       Home ID       |
|      (4 Octets)     |
+---------------------+
|    Source NodeID    |
|      (1 Octets)     |
+---------------------+
|    Frame Control    |
|      (2 Octets)     |
+---------------------+
| Destination NodeID  |
|     (0-1 Octets)    |
+---------------------+
|       Length        |
|      (1 Octet)      |
+---------------------+
|      Sequence       |
|      (1 Octet)      |
+---------------------+
|       Payload       |
|    (0-158 Octets)   |
+---------------------+
|        FCS          |
|     (2 Octes)       |
+---------------------+

Description

Z-Wave is defined by ITU-T Recommendation G.9959. The MAC header format here is used by Type 3 devices with a supported data rate of 100 Kbps (R3); some fields have been moved from their locations in the MAC header for Channel Configuration 3, as specified by G.9959.
